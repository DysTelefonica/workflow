# Skills del Framework

Cada carpeta dentro de este directorio es una skill independiente.

Estructura de una skill:

skill-name/
    SKILL.md
    references/
    scripts/

La IA debe descubrir automáticamente todas las skills disponibles
explorando este directorio.

Las skills principales del framework son:

- sdd-protocol
- spec-writer
- prd-writer
- rfc-writer
- hotfix
- access-vba-sync
